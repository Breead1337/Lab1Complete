# TravelManager

Приложение для учёта расходов в путешествиях (Windows Forms, .NET 6).

## Запуск

Открыть TravelManager.csproj в Visual Studio, собрать и запустить.

## Структура

- `Trip.cs` — класс поездки
- `Expense.cs` — класс расхода
- `TripForm.cs` — главная форма
- `AddExpenseForm.cs` — форма добавления расхода
- `TripReportForm.cs` — форма отчёта
- `Program.cs` — точка входа
